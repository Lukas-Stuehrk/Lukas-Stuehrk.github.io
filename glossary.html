Glossary

My website is using a lot of technical terms and jargon. This is a small collection of some of the terms that appear often and a short explanation.
CSS
Cascading Style Sheets. It's the style sheet language used to style websites.
HTML
Hypertext Markup Language. It's the markup language used to author websites.
CSV
Comma-Separated Values, a file format to store tabular data.
